# dotfiles

personal configuration files.

---

## major requirement

- OS:MacOS
- vim:NVIM v0.2.0 or later

NOTE:

  other tools like git, tmux, and else are not specified here due to the amount of them.

## correspondance of each configuration files

- shell

	.zshrc

	.zprofile

- vim

	.vimrc

	.dein.toml

	.dein_lazy.toml

- other CLI tools

	.git	<- git,github

	.tmux.conf	<- tmux

	config.json	<- peco
